What Does It Do?

It is a small package that combines two basic tasks into one:

  1. Write documents in R Markdown and convert them to HTML (R Markdown v1).
  2. Convert HTML with inlined CSS (juice).

Motivation

There has been a cost-saving trend in my university and they recommend instructors against printing teaching material. Also, students also bring their own PCs to the university. Distributing course content in digital format is becoming common. While distributing in PDF format is a preferred approach, nowadays many students do not print on paper, so I think it‚Äôs convenient to distribute important materials in HTML format that can be easily viewed on smartphones1.

At my university, we use Moodle as the learning management system (LMS), in which we can create web pages with ‚ÄúPage‚Äù module. You can directly edit HTML code and do various things, but if you write it like this:2

<style>
.red { color: red; }
</style>

<span class = "red">This is red</span>

the <style></style> is automatically removed, and the style information is not reflected. Therefore, you need to write style attributes directly inside each element as follows:

<span style = "color: red;">This is red</span>

Outside of user content, the site-wide CSS is applied so this design is necessary for safety reasons. There‚Äôs no way around it, but it can be cumbersome. Of course, for simple cases like the example above, it‚Äôs fine, but when HTML/CSS becomes complex, it would be impossible to do it manually.

That‚Äôs where using a node.js library called juice to inline style definitions comes in handy. As an R wrapper, there‚Äôs a package called juicyjuice.

You can perform the entire process of writing R Markdown and converting it to HTML, including inlining CSS, in R. This is what the juicedown package does.

At the drafting stage, specify full_html = TRUE and it will generate a complete HTML file. You can check the result by opening the ‚Äúfile.html‚Äù in your browser.

To change the design, you can pass the path to a CSS/SCSS file to the stylesheet option.
